Название проекта: Vage_dem

Описание:
Vage_dem — это демонстрационный проект, включающий функции регистрации, авторизации, а также добавления, удаления и редактирования заявок.

Функции:
1. Регистрация пользователей.
2. Авторизация пользователей.
3. Добавление заявок.
4. Удаление заявок.
5. Редактирование заявок.

Требования:
1. JDK версии 17.
2. Библиотеки:
   - MySQL Connector
   - Spring JDBC
3. База данных MySQL, настроенная через MySQL Workbench.

Установка:
1. Установите JDK 17.
2. Установите MySQL Workbench и настройте базу данных.
3. Склонируйте репозиторий проекта: `git clone <URL вашего репозитория>`.
4. Импортируйте проект в вашу IDE.
5. Настройте подключение к базе данных в конфигурационном файле проекта.
6. Скачайте и добавьте необходимые библиотеки (MySQL Connector и Spring JDBC) в проект.

Импорт базы данных:
1. Откройте MySQL Workbench.
2. Импортируйте готовый дамп базы данных:
   - Перейдите в меню "Файл" -> "Импортировать данные".
   - Выберите файл дампа базы данных `vage_dem_db.sql` и следуйте инструкциям на экране для завершения импорта.

Запуск:
1. Откройте проект в вашей IDE.
2. Запустите главный класс приложения.
3. Используйте интерфейс для регистрации, авторизации и управления заявками.
